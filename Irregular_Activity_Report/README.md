# r/place 2022 — Irregular Activity Analysis (Pixel Warfare, Automation Signals, and User Surges)

This project analyzes the 2022 **r/place** canvas to identify and characterize **irregular behavior patterns** that deviate from typical human participation. I was tasked to group any findings I made into “buckets” that represent distinct irregular activity types, then supported each with quantitative evidence (tables + plots) and reproducible scripts.
